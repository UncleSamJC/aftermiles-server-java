🚗 Traccar API 测试工具

⚠️ 认证说明

注意: 大部分API需要认证。首次使用需要:

  1. 先注册一个管理员用户 (通过 POST /api/users)
  2. 使用用户名密码登录获得会话
  3. 或使用Basic Authentication

📊 服务器信息

GET /api/server

获取服务器基本信息和配置

🔐 会话管理

POST /api/session

用户登录 (需要用户名和密码)

GET /api/session

获取当前会话信息

DELETE /api/session

用户登出

👥 用户管理

GET /api/users

获取用户列表 (需要管理员权限)

POST /api/users

创建新用户

📱 设备管理

GET /api/devices

获取设备列表

POST /api/devices

添加新设备

📍 位置数据

GET /api/positions

获取位置数据

📋 事件数据

GET /api/events

获取事件列表

💰 费用管理

GET /api/expenses

获取费用记录列表

POST /api/expenses

创建费用记录

🔧 开发工具推荐

对于更高级的API测试，推荐使用:

  • Postman: 功能强大的API测试工具
  • Insomnia: 轻量级的REST客户端
  • VS Code REST Client: VS Code插件
  • curl命令: 命令行工具
